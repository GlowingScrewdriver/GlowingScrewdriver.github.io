## About this Website
This website is a running experiment of mine, written by hand without using any of the heavy frameworks/toolkits
that crowd the web today. The goal is to have a website that is fast to load, small in size, and does as little as possible during
runtime. Neither do I need JavaScript to generate content that looks the same each time, nor do I need kilobytes together of CSS for
content that is all text and images.

There is a build/deployment process though (I am, after all, a human. I don't speak XML). The only dependencies pulled in are:

* GNU Make
* Markdown (yes, the OG Markdown)
* A POSIX Shell (no, it doesn't have to be Bash)
* (Were you expecting more? Okay, I also used some standard Unix utilities)
